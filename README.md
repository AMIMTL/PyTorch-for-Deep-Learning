## Course Overview
This bootcamp teaches deep learning with Facebook's PyTorch library. It balances important theoretical concepts with hands-on exercises and projects, covering everything from NumPy and Pandas to state-of-the-art neural networks.

## What You'll Learn
- Use NumPy for array formatting and Pandas for data manipulation
- Understand classic machine learning theory (train/test splits, model evaluation, unsupervised learning)
- Work with PyTorch Tensors
- Build Artificial Neural Networks (ANNs)
- Create Convolutional Neural Networks (CNNs) for image classification
- Implement Recurrent Neural Networks (RNNs) for sequence/time series data
- Build deep learning models for tabular data

## Prerequisites
- Basic Python (data types, loops, functions)
- Python OOP recommended
- Basic derivative calculations
- Admin permissions to install software
